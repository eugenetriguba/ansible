# Dotfiles

![Preview](/docs/preview.png)

Personal dotfiles to get me up and running on xubuntu.

## Getting Started

Install the dotfiles.
```bash
$ ./bootstrap.py
```

Uninstall
```bash
$ ./bootstrap.py --uninstall
```
